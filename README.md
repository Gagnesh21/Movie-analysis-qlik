# Movie Analytics Dashboard (Qlik Sense)

## Project Overview
This project analyzes key drivers of movie profitability and global revenue using Qlik Sense. 
The objective is to identify which factors most strongly influence financial success in the film industry.

## Dataset
The dataset includes:
- Budget
- Opening Weekend Revenue
- Global Revenue
- Genre
- Studio
- Year of Release

Total records: 971 movies

## Key Business Questions
1. Does opening weekend performance predict total global revenue?
2. How does budget impact profitability?
3. Which genres and studios generate the highest returns?
4. Does release scale influence revenue concentration?

## Key Insights
- Opening weekend performance is the strongest predictor of global revenue.
- Higher release scale generally correlates with higher revenue.
- Certain studios consistently outperform in profitability.
- Profitability varies significantly across genres.
